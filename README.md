# Sales Performance Analytics Dashboard

## Overview

Analyzed retail sales data using Python to identify trends, regional performance, and top-selling product categories.

* Pandas

* Seaborn
* Google Colab

## Key Features

* Sales Trend Analysis
* Regional Performance Analysis
* Product Category Analysis
* Data Visualization

## Skills Demonstrated

* Data Analytics
* Exploratory Data Analysis (EDA)
* Python Programming
* Data Visualization
